Instrucciones de uso
  1. El Visualizador permite ver todas las partes del destornillador desde cualquier ángulo.
  2. Selecciona cualquier punto del cuerpo y arrastra para girar el objeto.
  3. Para mover piezas, usa el menú "Mover piezas" (Gatillo, Mandril, Broca, encender/apagar).
  4. La opción animación es para modelar el objeto (opcional).
